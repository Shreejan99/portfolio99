# Personal Portfolio

This is my personal portfolio website showcasing my skills, projects, and contact information.  
It also includes a Download Resume button linking to my resume hosted on Azure Blob Storage.

## How to Run Locally

1. Clone the repository:
2. 2. Open `index.html` in your browser.

## Deploy on Vercel

1. Go to [Vercel](https://vercel.com) and sign in with GitHub.  
2. Click **New Project** → select this repository.  
3. Framework Preset → choose **Other**.  
4. Click **Deploy** → your portfolio will be live.  

## Update Resume

- Upload your resume to Azure Blob.  
- Copy the Blob URL.  
- Replace the URL in `index.html` for the Download Resume button.
